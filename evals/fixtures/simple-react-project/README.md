# Simple React Project

A minimal React + Vite + TypeScript project used for testing harness-context-engine.

## Quick Start

```bash
npm install
npm run dev
```

## Available Scripts

- `dev`: Start Vite dev server
- `build`: TypeScript compile + Vite build
- `preview`: Preview production build
- `lint`: ESLint check

## Project Structure

```
src/
├── App.tsx      # Main app component (counter demo)
├── main.tsx     # Entry point
├── components/  # Shared components (empty)
```

## Framework

- React 18
- TypeScript 5
- Vite 5
